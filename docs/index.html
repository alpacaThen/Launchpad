Installation Help

Having trouble opening Launchpad? Here's how to fix the "cannot verify developer" warning.

‚ö†Ô∏è
Seeing this message?

"Launchpad cannot be opened because the developer cannot be verified"

1

Open System Settings

Click the Apple menu () in the top-left corner and select System Settings (or System Preferences on older macOS versions).

2

Go to Privacy & Security

In System Settings, click on Privacy & Security in the sidebar.

3

Allow Launchpad to Open

Scroll down to the Security section. You'll see a message about Launchpad being blocked. Click "Open Anyway" and confirm with your password.

Alternative Method: Right-Click to Open

  1. Locate the Launchpad app in Finder
  2. Right-click (or Control-click) on the app icon
  3. Select "Open" from the menu
  4. Click "Open" again in the dialog that appears
üí°
Why does this happen?

macOS Gatekeeper protects your Mac by checking if apps are from identified developers. Since Launchpad is not distributed through the Mac App Store, you need to explicitly allow it to run. This is a one-time process.
